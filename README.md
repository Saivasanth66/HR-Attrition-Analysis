# HR Attrition Analysis 📊

## 📌 Project Overview
This project analyzes employee attrition data using Python and Pandas to identify turnover trends and risk factors.

## 🚀 Features
- Calculate attrition rate
- Department-wise attrition analysis
- Salary comparison (Stayed vs Left)
- Experience analysis
- Automated Excel report generation

## 🛠 Tech Stack
- Python
- Pandas
- Excel

## 📂 Project Structure
data/ → Dataset  
src/ → Analysis script  
output/ → Generated reports  

## ▶️ How to Run
1. Install requirements  
   pip install -r requirements.txt  

2. Run the script  
   python src/analysis.py  

## 📊 Sample Output
- Attrition Rate
- Department-wise turnover
- Salary & Experience comparison

## 🎯 Business Impact
Helps HR teams identify high-risk departments and understand factors contributing to employee turnover.
